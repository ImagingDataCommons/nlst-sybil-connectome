NLST Sybil Tumor Connectome

Ready
Content-based image retrieval (CBIR) connectome for the National Lung Screening Trial (NLST) dataset. Each node is a patient; edges connect to the 5 nearest (or farthest) matches in the embedding space of 9 foundation models. Imaging data from NCI Imaging Data Commons (IDC). Tumor annotations from NLST-Sybil. Built with Claude Code using the IDC skill. Source code on GitHub.
